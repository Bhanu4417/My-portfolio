# Bhanu's Portfolio

Welcome to the source code of my personal portfolio website!

This is a modern, highly interactive, and beautifully designed web application built to showcase my skills, projects, and live coding statistics.

## ✨ Features
- **Dynamic Theming**: Seamless dark and light mode toggle.
- **Live Coding Stats**: Real-time integration with my WacheIt and GitHub data.
- **Interactive Graphs**: Custom-built contribution graphs.
- **Modern UI/UX**: Premium aesthetic with micro-animations, glassmorphism, and Tailwind CSS.

## 🛠️ Tech Stack
- **Framework**: [Next.js 15 (App Router)](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)

## 🚀 Running Locally

1. Clone the repository
```bash
git clone https://github.com/Bhanu4417/My-portfolio.git
cd My-portfolio
```

2. Install dependencies
```bash
npm install
```

3. Run the development server
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
